How to run?

*Is this the first time that you want to run CDM?
run the following command in 2 directories './backend/' and './gui/':
npm install

For running the app, use the following command in './gui/':
npm run serve

**MAKE SURE THAT YOU INSTALLED NPM FOR THAT 2 DIRECTORIES BEFORE**
